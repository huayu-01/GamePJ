# 局域网连接说明

游戏使用 UDP 7000 传输对局，UDP 17341 搜索房间。

## 实体设备

- Host 与加入者必须位于能够互相访问的同一局域网。
- 路由器需关闭“AP 隔离”“客户端隔离”或“访客网络隔离”。
- Windows Host 以管理员 PowerShell 运行 `tools/network/Enable-GamePJ-Firewall.ps1`。
- 手机 Host 显示的地址应由另一设备直接填写，例如 `192.168.1.9:7000`。

## Android Emulator

模拟器地址 `10.0.2.15` 属于私有 NAT，实体手机无法直接连接该地址，也无法通过普通局域网广播发现它。

- 模拟器加入电脑 Host：填写 `10.0.2.2:7000`。
- 实体手机互测：应使用两台实体设备，或让电脑/手机创建可被同网设备访问的房间。
- 若必须让实体手机加入模拟器 Host，需要额外配置模拟器 UDP 端口重定向和 Windows 防火墙；这只适合开发测试。

## 快速诊断

电脑无法 `ping` 或解析到手机的 MAC 地址时，问题发生在 Godot 之前。先确认手机没有使用移动数据、双方没有接入访客网络，并关闭路由器客户端隔离。
